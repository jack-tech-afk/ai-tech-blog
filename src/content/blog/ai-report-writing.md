本記事では、**ChatGPTとClaudeを活用して報告書作成を劇的に効率化**する方法を、具体的なプロンプト例とともに解説します。

| 報告書の種類 | 従来の作成時間 | AI活用後 | 短縮率 |
|-------------|-------------|---------|--------|
| 日報 | 20分 | 5分 | 75% |
| 週報 | 45分 | 10分 | 78% |
| 月次報告 | 3時間 | 30分 | 83% |

## 日報をAIで5分で書く方法

### 箇条書きメモ → 完成日報のプロンプト

仕事中にメモした箇条書きを、AIに投げるだけで日報が完成します。

**プロンプト例：**

```
以下の箇条書きメモを社内日報の形式に整えてください。

【今日のメモ】
- A社との定例MTG 10:00-11:00、次回提案書持参
- B案件の見積もり修正、15%値引き対応
- 新人研修のスライド20枚作成
- C社から問い合わせ対応（納期確認）
- 来週の出張手配完了（大阪、3/24-25）

【フォーマット】
1. 本日の業務内容（箇条書き・時系列）
2. 進捗・成果
3. 課題・懸念事項
4. 明日の予定
```

たったこれだけで、フォーマットに沿った日報が10秒で生成されます。

## 週報・月次報告の効率化テクニック

### 週報の作成プロンプト

```
以下の今週の業務メモから、週次報告書を作成してください。

【今週の業務メモ】
[月〜金の主要業務を箇条書きで入力]

【週報フォーマット】
1. 今週のハイライト（重要トピック3つ）
2. 各案件の進捗状況（テーブル形式: 案件名/ステータス/進捗率/次のアクション）
3. KPI達成状況
4. 来週の注力事項
5. 上長への相談・報告事項
```

### 月次報告の作成プロンプト

月次報告は分量が多いため、**Claudeの長文処理能力**が活きます。

```
以下のデータから月次報告書を作成してください。

【月次データ】
- 売上: [金額]（前月比: [%]）
- 新規顧客数: [件]
- 主要プロジェクト進捗: [箇条書き]
- 課題・リスク: [箇条書き]
- 来月の計画: [箇条書き]

【要件】
- エグゼクティブサマリーを冒頭に配置
- 前月との比較を含む
- 課題には対策案を併記
- 図表用のテーブルデータも含める
```

## 報告書AI活用の3つの注意点

### 1. 「AIが書いた」ことが透けないようにする

AIの文章には特徴的なパターンがあります（「〜と言えるでしょう」の多用など）。最終的に自分の言葉でリライトすることで、自然な報告書に仕上がります。

### 2. 数値データは必ず自分で確認する

AIは計算ミスや架空の数値を出力することがあります。売上データ、KPI、予算など**数値に関わる部分は必ずダブルチェック**してください。

### 3. フォーマットを登録して再利用する

毎回プロンプトを打つのは面倒です。以下の方法でフォーマットを保存しましょう。

- **ChatGPT** → カスタム指示（Custom Instructions）に報告書フォーマットを登録
- **Claude** → プロジェクト機能にフォーマットをプリセット
- **Notion AI** → テンプレートとして保存
