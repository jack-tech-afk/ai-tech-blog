アンケートを実施したものの、**回答データの集計・分析に時間がかかって困っていませんか？**

特に自由記述の回答は、手作業で分類すると膨大な時間がかかります。500件の自由記述を一つずつ読んで分類する——想像するだけで気が重くなりますよね。

**ChatGPTとExcelを組み合わせれば、アンケート分析を大幅に自動化**できます。この記事では、データの前処理からレポート作成まで、AI活用の全手順を解説します。

| ステップ | 作業内容 | ツール | 所要時間 |
|---------|---------|-------|---------|
| ①データ前処理 | 不要データの削除・形式統一 | Excel | 10分 |
| ②定量分析 | 選択式回答の集計・クロス集計 | Excel / ChatGPT | 15分 |
| ③定性分析 | 自由記述の分類・感情分析 | ChatGPT | 20分 |
| ④レポート作成 | グラフ作成・考察・提案 | ChatGPT + Excel | 15分 |

従来半日かかっていた分析作業が、**約1時間で完了**します。

## ステップ①：データの前処理

### Excel / スプレッドシートで整形

アンケートデータをAIに渡す前に、以下の前処理を行います。

1. **個人情報の削除**：氏名・メールアドレス等を削除またはマスキング
2. **空白行の削除**：未回答のレコードを処理
3. **表記ゆれの統一**：「男」「男性」「Male」→「男性」に統一
4. **CSVエクスポート**：ChatGPTにアップロードする形式に変換

Excel Copilotを使えば、「表記ゆれを統一して」と指示するだけで自動的に修正してくれます。

---

## ステップ②：選択式回答の定量分析

### 基本集計をAIで自動化

CSVファイルをChatGPTにアップロードし、以下のプロンプトで分析させます。

```
添付のアンケートデータを分析してください。

以下を実行してください：
1. 各質問の回答分布（割合）を表で表示
2. 特徴的な傾向を3つ指摘
3. 全体の回答傾向をグラフで可視化

注意：
- 日本語で回答してください
- 割合は小数点第1位まで表示
```

ChatGPTのAdvanced Data Analysis機能がPythonコードを自動生成し、集計結果とグラフを出力してくれます。

### クロス集計で深い分析

単純集計だけでは見えない傾向を、クロス集計で発見しましょう。

```
以下のクロス集計を実行してください：
1. 「年代」×「満足度」のクロス集計表
2. 「利用頻度」×「NPS（推奨度）」の相関
3. 「職種」×「改善要望」の傾向分析

統計的に有意な差がある組み合わせをハイライトしてください。
```

## ステップ③：自由記述の自動分類と感情分析

### テキスト分類プロンプト

自由記述の回答をAIで自動分類する方法です。

```
以下のアンケート自由記述データを分析してください。

1. 回答を以下のカテゴリに自動分類してください
   - 機能に関する要望
   - 価格に関する意見
   - サポートに関する評価
   - UIに関する意見
   - その他

2. 各カテゴリの回答数と割合を表で表示

3. 各カテゴリの代表的な意見を3つずつ抽出

データ：
[自由記述データを貼り付け]
```

### 感情分析（センチメント分析）

```
各自由記述の感情を分析してください。

分類：
- ポジティブ（満足・感謝・推奨）
- ネガティブ（不満・批判・改善要望）
- ニュートラル（事実の記述・質問）

出力形式：
| 回答ID | 回答内容（要約） | 感情 | 信頼度 |
```

AIの感情分析は、手作業と比較して**精度85〜90%**程度と高い水準を実現しています。大量データの「一次スクリーニング」として非常に有効です。

## ステップ④：レポートの自動作成

### 分析レポートの自動生成

集計結果を基に、報告用のレポートを自動生成させます。

```
これまでの分析結果を基に、以下の構成で分析レポートを作成してください。

1. エグゼクティブサマリー（3行以内）
2. 調査概要（目的・期間・回答者数・方法）
3. 主要な発見（Top 5）
4. 定量分析の結果（表・グラフ付き）
5. 定性分析の結果（自由記述の傾向）
6. 改善提案（優先度付きで3〜5項目）
7. 次のステップ

対象読者：経営層
トーン：簡潔で根拠に基づいた表現
```

### Excelでグラフを仕上げる

ChatGPTが生成したグラフはそのまま使える品質ですが、社内テンプレートに合わせたい場合はExcelで仕上げましょう。ChatGPTに「Excel用のデータをCSVで出力して」と指示すれば、すぐにExcelに取り込めます。

## 注意点とベストプラクティス

### データプライバシーの重要性

- 個人情報は必ず削除してからAIに渡す
- 企業の機密情報を含むデータはChatGPT Enterpriseを利用
- 分析後は不要なチャット履歴を削除

### AI分析の限界を理解する

- AIの分析結果は「示唆」であり「結論」ではない
- 統計的な有意差検定は別途確認が必要
- 回答者のバイアス（選択バイアス等）はAIでは検出できない

## まとめ：AIでアンケート分析を10倍速にする

AIアンケート分析のポイントをまとめます。

- **データ前処理**はExcelで整形し、個人情報は必ず削除
- **定量分析**はChatGPTのAdvanced Data Analysisで自動集計
- **自由記述**はAIの感情分析・テキスト分類で大幅時短
- **レポート**もAIで自動生成し、最終調整は人間が行う

アンケートを「実施して終わり」にせず、AIの力で素早く分析し、ビジネスの改善に活かしましょう。
