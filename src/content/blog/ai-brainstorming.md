企画会議で「何かいいアイデアない？」と聞かれて、頭が真っ白になった経験はありませんか？

ブレインストーミングが苦手な人でも、**AIを壁打ち相手に使えば、10分で50個以上のアイデア**を量産できます。しかもAIは「それ微妙だね」と否定しないので、心理的安全性も完璧です。

この記事では、ChatGPTやClaudeを使った**AIブレインストーミングの実践手法5選**と、そのまま使えるプロンプトテンプレートを紹介します。

## AIブレインストーミングとは？

AIブレインストーミングとは、ChatGPTやClaudeなどの生成AIを**アイデア発想の壁打ち相手**として活用する手法です。従来のブレストと異なり、以下のメリットがあります。

- **1人でもできる** — チームを集める必要がない
- **24時間対応** — 深夜のひらめきにも即対応
- **心理的安全性** — 突飛なアイデアでも否定されない
- **大量生成** — 10分で50〜100個のアイデアを出せる
- **多角的視点** — 異なる業界・立場からの発想が可能

## AIブレスト手法5選

### 手法1：「100個リスト」で量を出す

最もシンプルな手法です。テーマを指定して、AIに大量のアイデアを列挙させます。

**プロンプト例：**
```
あなたは創造的な企画コンサルタントです。
「30代向けのサブスクリプションサービス」のアイデアを
50個、箇条書きで列挙してください。
ジャンルは問いません。突飛なアイデアも歓迎します。
```

ポイントは**「突飛なアイデアも歓迎」**と明記すること。AIは安全なアイデアに寄りがちなので、意図的にハードルを下げましょう。

### 手法2：SCAMPER法 × AI

SCAMPER法は、既存のアイデアを7つの視点で変形する古典的な発想法です。AIと組み合わせると圧倒的に効率が上がります。

| 視点 | 意味 | 質問例 |
|------|------|--------|
| **S** Substitute | 代替する | 何を別のものに置き換えられるか？ |
| **C** Combine | 組み合わせる | 何と何を合体させられるか？ |
| **A** Adapt | 適応する | 他の業界から応用できるか？ |
| **M** Modify | 修正する | サイズ・形・機能を変えたら？ |
| **P** Put to other use | 転用する | 別の用途に使えないか？ |
| **E** Eliminate | 排除する | 何を取り除けるか？ |
| **R** Reverse | 逆転する | 逆にしたらどうなるか？ |

**プロンプト例：**
```
「オフィスのコーヒーサーバー」をSCAMPER法の7つの視点で
それぞれ3つずつアイデアを出してください。
各アイデアには実現可能性（高/中/低）も付けてください。
```

### 手法3：ペルソナ切り替え法

AIに異なるペルソナ（役割）を演じさせて、多角的な視点からアイデアを引き出します。

**プロンプト例：**
```
「社員食堂のリニューアル企画」について、
以下の5つの立場からそれぞれアイデアを3つ出してください。

1. 20代の新入社員
2. 子育て中の40代マネージャー
3. 健康オタクのCTO
4. コスト削減を求める経理部長
5. ミシュラン星付きシェフ
```

同じテーマでも立場が変わるとまったく異なるアイデアが生まれます。

### 手法4：「もしも」シナリオ法

制約を外した仮定の質問で、常識にとらわれないアイデアを引き出します。

**プロンプト例：**
```
もし以下の条件が実現したら、私たちのビジネスはどう変わりますか？
それぞれ5つのビジネスチャンスを考えてください。

1. もし全人類がテレパシーを使えたら
2. もし移動コストがゼロになったら
3. もし睡眠が不要になったら
4. もし言語の壁が完全になくなったら
```

突飛な仮定からも、意外と実現可能なアイデアの種が見つかります。

### 手法5：反転ブレスト法

「良いアイデア」ではなく「最悪のアイデア」を先に出してから、それを反転させます。

**プロンプト例：**
```
ステップ1: 「顧客満足度を最低にする方法」を10個考えてください。
ステップ2: それぞれを反転させて「顧客満足度を最高にする方法」にしてください。
ステップ3: 反転したアイデアの中で、特に実行しやすいものTOP3を教えてください。
```

ネガティブな発想から入ることで、普段気づかない改善ポイントが浮かび上がります。

## 企画会議での活用フロー

### 会議前（15分）：AIでたたき台を生成

1. テーマを設定し、手法1〜5のプロンプトを実行
2. AIが生成した50〜100個のアイデアをスプレッドシートにまとめる
3. 明らかに非現実的なものを除外し、30個程度に絞る

### 会議中（30分）：AIたたき台をベースに議論

1. 30個のアイデアリストを参加者に共有
2. 各自が気になるアイデアを3つ選ぶ（投票）
3. 上位5つについて深掘りディスカッション

### 会議後（10分）：AIで具体化

1. 選ばれたアイデアをAIで具体化する
2. 「このアイデアの実現に必要なステップを5つ教えて」と指示
3. 実行計画のドラフトを自動生成

## ChatGPT vs Claude：ブレストでの使い分け

| 観点 | ChatGPT | Claude |
|------|---------|--------|
| アイデアの量 | ◎ 多い | ○ やや少ない |
| アイデアの質 | ○ 幅広い | ◎ 論理的 |
| 深掘り力 | ○ | ◎ |
| 視覚的出力 | ○ | ◎（Artifacts） |
| 長文処理 | ○ | ◎（20万トークン） |

**おすすめの使い分け：**
- 大量にアイデアを出したい → ChatGPT
- 各アイデアを論理的に深掘りしたい → Claude
- 最強の組み合わせ：ChatGPTで量を出す → Claudeで質を高める
