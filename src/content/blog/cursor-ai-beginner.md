## Cursorとは？AIがコーディングを助けてくれるエディタ

**Cursor（カーソル）**は、AIを搭載したプログラミング用コードエディタです。VS Code（Visual Studio Code）をベースに作られており、見た目や操作感はVS Codeとほぼ同じ。しかしAI機能がエディタに深く統合されている点が大きく異なります。

2026年現在、CursorはAIコーディングエディタの中で**最も人気のあるツール**の一つとなっており、個人開発者からプロのエンジニアまで幅広く利用されています。

---

## CursorとVS Codeの違い

| 項目 | Cursor | VS Code |
|------|--------|---------|
| **AIチャット** | 内蔵（Cmd+L） | 拡張機能で追加 |
| **AI補完** | 高精度なAI補完が標準 | Copilot拡張が必要 |
| **AIコード編集** | Cmd+Kで直接AI編集 | 非対応 |
| **コードベース理解** | プロジェクト全体をAIが把握 | ファイル単位のみ |
| **拡張機能** | VS Code互換 | 豊富なエコシステム |
| **料金** | 無料〜月額20ドル | 完全無料 |

ポイントは、**VS Codeの使い勝手はそのままに、AIがコーディングパートナーとして常に隣にいる**感覚で開発できることです。

## Cursorのインストール方法

### Step 1: ダウンロード
[Cursor公式サイト](https://cursor.com/)にアクセスし、「Download」ボタンをクリック。Mac・Windows・Linuxに対応しています。

### Step 2: インストール
ダウンロードしたファイルを実行してインストール。Macの場合はdmgファイルを開いてApplicationsフォルダにドラッグするだけです。

### Step 3: VS Codeの設定を移行
初回起動時に「Import VS Code Settings」の選択肢が表示されます。VS Codeを使っていた方は、ここで設定・拡張機能・キーバインドをワンクリックで移行できます。

### Step 4: アカウント作成
メールアドレスまたはGoogleアカウントでサインアップ。無料プランですぐに使い始められます。

## Cursorの基本機能3つ

### 1. AIチャット（Cmd + L）

エディタの右側にAIチャットパネルが開き、**コードについて何でも質問**できます。

**使い方の例：**
- 「この関数の処理内容を説明して」
- 「このエラーの原因と修正方法は？」
- 「ユーザー認証機能のコードを書いて」

コードを選択した状態で`Cmd+L`を押すと、選択したコードがチャットに自動で添付され、そのコードについて具体的に質問できます。

### 2. インラインAI編集（Cmd + K）

コードの一部を選択して`Cmd+K`を押すと、**AIがその場でコードを書き換え**てくれます。

**使い方の例：**
- 関数を選択 →「エラーハンドリングを追加して」
- ループ処理を選択 →「これをmap関数に書き換えて」
- コメントを書いて選択 →「このコメントの通りに実装して」

変更内容はdiff表示で確認でき、「Accept」で適用、「Reject」でキャンセルできます。

### 3. Tab補完（AI自動補完）

コードを書いている途中で、**AIが次に書くべきコードを予測して提案**してくれます。Tabキーを押すだけで提案を受け入れられます。

通常のオートコンプリートとは異なり、**複数行にわたるコードブロック**や**文脈に応じた関数全体**を提案してくれるのが特徴です。

## 初心者がCursorを活用するコツ

### コツ1: まずは「質問」から始める
コードを書くのではなく、**AIに質問することから始めましょう**。「Pythonでファイルを読み込む方法は？」のような基本的な質問でもOKです。

### コツ2: エラーメッセージをそのままAIに投げる
エラーが出たら、エラーメッセージをコピーしてAIチャットに貼り付けるだけ。原因と解決策を教えてくれます。

### コツ3: 日本語で指示してOK
CursorのAIは**日本語の指示を正確に理解**します。英語が苦手でも問題ありません。
