「毎日の日報作成に15分もかかっている」「週報をまとめるのが面倒で金曜日が憂鬱」——多くのビジネスパーソンが抱えるこの悩み、**AIで解決**できます。

ChatGPTを使えば、箇条書きのメモから**わずか数分で読みやすい日報・週報**が完成します。この記事では、テンプレートの作り方からSlack連携の自動化まで、具体的な手順を解説します。

## 日報・週報作成にAIが最適な理由

日報作成は以下の特徴があるため、AI活用に最も向いている業務の一つです。

- **定型フォーマット**: 毎日同じ構成で書く → テンプレート化しやすい
- **箇条書き→文章変換**: メモを整形する単純作業 → AIの得意分野
- **繰り返し作業**: 毎日・毎週発生 → 自動化の効果が大きい
- **文体の統一**: 会社のトーンに合わせる → AIに学習させられる

**時間短縮の目安：**

| 作業 | 従来 | AI活用後 |
|------|------|---------|
| 日報作成 | 15〜30分 | 3〜5分 |
| 週報作成 | 30〜60分 | 5〜10分 |
| 月報作成 | 1〜2時間 | 15〜30分 |

## 方法1：ChatGPTで日報を生成する

### ステップ1：テンプレートプロンプトを作る

以下のプロンプトをChatGPTに入力すると、日報専用のアシスタントとして動作します。

```
あなたは日報作成アシスタントです。
私がメモを入力したら、以下のフォーマットで日報を作成してください。

【日報フォーマット】
■ 日付: YYYY/MM/DD（曜日）
■ 本日の業務内容:
  1. [業務名] - [詳細・成果]
  2. [業務名] - [詳細・成果]
■ 進捗状況:
  - [プロジェクト名]: [進捗率]% [状況]
■ 課題・懸念事項:
  - [内容]
■ 明日の予定:
  - [予定内容]
■ 所感:
  [1〜2文で所感を記載]

ルール:
- 敬体（です・ます調）で記載
- 具体的な数値を含める
- 簡潔かつ読みやすい文章にする
```

### ステップ2：メモを入力する

```
・午前：A社向け提案書作成 8割完了
・午後：B社とオンライン商談 好感触
・C案件のスケジュール遅延 1週間
・明日：提案書仕上げ、D社訪問準備
```

### ステップ3：AIが日報を生成

AIが上記のメモを整形し、フォーマットに沿った日報を出力します。所感まで自動で追加されるため、コピペするだけで提出できます。

## 方法2：GPTsで日報専用ボットを作る

ChatGPT Plusのユーザーは、**GPTs（カスタムGPT）**で自社専用の日報ボットを作れます。

### 設定手順

1. **ChatGPT → 「GPTを作成」を選択**
2. **名前**: 「日報アシスタント」
3. **指示内容**に以下を設定：

```
あなたは[会社名]の日報作成アシスタントです。

役割:
- ユーザーのメモから日報を生成する
- 会社のフォーマットに準拠する
- 簡潔で読みやすい文章を書く

フォーマット:
[自社の日報フォーマットを貼り付け]

トーン:
- ビジネス敬語
- 具体的数値を含める
- ポジティブな表現を心がける
```

4. **保存してチーム内で共有**

### GPTsのメリット

- **毎回プロンプトを入力する必要がない**
- **チーム内で統一フォーマットを共有できる**
- **過去の会話履歴から文体を学習**

## 方法3：週報を日報から自動集約する

毎日の日報を週報にまとめる作業もAIで自動化できます。

### プロンプト例

```
以下は今週（月〜金）の日報です。
これらを週報にまとめてください。

【週報フォーマット】
■ 期間: MM/DD（月）〜MM/DD（金）
■ 今週の主な成果（3〜5項目）:
■ プロジェクト別進捗:
■ 来週の重点タスク:
■ 課題・相談事項:
■ 週間所感:

[月曜の日報]
[火曜の日報]
...
[金曜の日報]
```

AIが5日分の日報を分析し、重複を除去しながら重要なポイントだけを抽出した週報を生成します。

## 方法4：Slack連携で完全自動化する

さらに効率化したい場合は、Slack × ChatGPT APIの連携がおすすめです。

### 自動化フロー

```
Slackにメモ投稿 → Zapier/n8n が検知 → ChatGPT APIで日報生成 → Slackに自動投稿
```

### Zapierでの設定手順

1. **トリガー**: Slack - New Message in Channel（#daily-memo）
2. **アクション1**: OpenAI - Send Prompt（日報生成プロンプト＋メモ内容）
3. **アクション2**: Slack - Send Message（#daily-report チャンネルに投稿）

### 必要なツール

| ツール | 用途 | 費用 |
|--------|------|------|
| Slack | メモ入力・日報出力 | 無料〜 |
| Zapier | 自動化フロー | 月額$19.99〜 |
| ChatGPT API | 日報生成 | 従量課金（月$1〜5程度） |

### 代替ツール

- **n8n**: オープンソースの自動化ツール（セルフホスト無料）
- **Make (Integromat)**: Zapierの代替（月額$9〜）
- **Notion AI**: Notionで日報管理している場合に最適

## まとめ：日報は書くものではなく生成するもの

2. **テンプレ化**: 自社フォーマットに合わせたプロンプトを作る
3. **GPTs化**: チーム共有の日報ボットを作成する
4. **自動化**: Slack連携で入力から提出まで自動化する
