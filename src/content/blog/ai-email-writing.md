ビジネスパーソンが1日にメールに費やす時間は**平均2.5時間**と言われています。メールの文面を考え、敬語を調整し、何度も見直す——この時間をAIで大幅に削減できます。

本記事では、**ChatGPTとCopilotを使ってビジネスメールを30秒で書く方法**を、シーン別テンプレート付きで解説します。

## AIメール作成の基本プロンプト

どんなメールでも使える**万能プロンプト**がこちらです。

```
以下の条件でビジネスメールを作成してください。

【宛先】[相手の役職・関係性]
【目的】[メールの目的]
【要点】[伝えたい内容を箇条書き]
【トーン】[丁寧/ややカジュアル/フォーマル]
【文字数】200〜300文字程度
```

この型に当てはめるだけで、あらゆるビジネスメールが瞬時に完成します。

## シーン別メールテンプレート8選

### 1. お礼メール

```
ビジネスメールを書いてください。
宛先：取引先の部長
目的：先日の打ち合わせのお礼
要点：
- 貴重なお時間をいただいたことへの感謝
- 提案内容を社内で検討する旨
- 来週中に回答予定
トーン：丁寧
```

### 2. 依頼メール

```
ビジネスメールを書いてください。
宛先：社内の他部署マネージャー
目的：資料提供の依頼
要点：
- A案件のプレゼン資料に使用するデータが必要
- 売上データ（2025年4月〜2026年3月）をExcelで共有希望
- 期限は3月21日（金）まで
トーン：丁寧だが社内なのでやや簡潔に
```

### 3. 謝罪メール

```
ビジネスメールを書いてください。
宛先：クライアントの担当者
目的：納期遅延のお詫び
要点：
- 当初3/15納品予定だったが3/22に変更
- 原因は品質チェックでの修正対応
- 対策として今後のチェック体制を強化
- お詫びの品として次回10%割引を提案
トーン：非常に丁寧、誠実
```

### 4. 催促メール

```
ビジネスメールを書いてください。
宛先：取引先の担当者
目的：見積もり回答の催促
要点：
- 3/10に送付した見積もりの回答をお願い
- 社内稟議の関係で3/19までに回答が必要
- ご不明点があればいつでも連絡してほしい
トーン：催促だが柔らかく、相手を責めない
```

### 5. 日程調整メール

```
ビジネスメールを書いてください。
宛先：社外のパートナー企業
目的：打ち合わせの日程調整
要点：
- 新プロジェクトのキックオフMTGの日程調整
- 候補日：3/24(火)午前、3/25(水)午後、3/26(木)終日
- 場所：オンライン（Zoom）
- 所要時間：1時間
トーン：丁寧
```

### 6. 断りメール

```
ビジネスメールを書いてください。
宛先：営業担当者
目的：提案のお断り
要点：
- 提案内容を社内で慎重に検討した
- 今回は予算の関係で見送り
- 提案自体は魅力的だった（フォロー）
- 次年度に改めて検討したい
トーン：丁寧、相手を不快にさせない
```

### 7. 社内連絡メール

```
ビジネスメールを書いてください。
宛先：チームメンバー全員
目的：プロジェクト進捗の共有
要点：
- フェーズ1完了（予定通り）
- フェーズ2は来週から開始
- 各自タスクの確認をお願い
- 不明点は明日の定例で質問OK
トーン：フレンドリーだがプロフェッショナル
```

### 8. フォローアップメール

```
ビジネスメールを書いてください。
宛先：商談相手
目的：商談後のフォローアップ
要点：
- 本日の商談のお礼
- 商談で質問いただいた点の回答（添付資料あり）
- 次のステップの提案
- デモ環境の提供も可能
トーン：丁寧かつ積極的
```

## Copilot in Outlookの活用法

Microsoft 365ユーザーなら、**Outlookに統合されたCopilot**が最強です。

### Copilotでできること

1. **新規メール作成** — 「〇〇について取引先にメールを書いて」と指示するだけ
2. **返信の自動生成** — 受信メールの内容を理解して返信案を提案
3. **トーンの調整** — 「もう少しフォーマルに」「簡潔にして」と微調整
4. **長文の要約** — 長いメールスレッドを一瞬で要約

### Copilotの最大のメリット

ChatGPTとの違いは、**メーラーの中で完結する**点です。ChatGPTの場合、生成→コピー→貼り付けの手間がありますが、Copilotなら**ワンクリックでメールに反映**されます。

## AIメール作成の5つのコツ

### 1. 箇条書きで要点を投げる

長い文章で説明するより、箇条書きで要点を伝えたほうがAIの出力品質が上がります。

### 2. 敬語レベルを指定する

「丁寧」「ややカジュアル」「非常にフォーマル」など、トーンを具体的に指定しましょう。

### 3. 文字数を指定する

「200文字程度」と指定すると適切な長さに収まります。指定しないと長くなりがちです。

### 4. カスタム指示を活用する

ChatGPTのカスタム指示に「私は〇〇株式会社の営業部の田中です。メールの署名は以下の通りです…」と登録しておくと、毎回の入力が省けます。

### 5. 定型メールはテンプレ保存

よく使うプロンプトはメモアプリに保存しておき、コピペで使い回しましょう。
