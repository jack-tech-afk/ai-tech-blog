## プラットフォーム別プロンプト集

### X（Twitter）向けプロンプト

**140字以内の投稿生成：**
```
あなたはSNSマーケティングの専門家です。
以下のテーマでX（Twitter）の投稿を5パターン作成してください。

テーマ: [テーマ]
ターゲット: [ターゲット層]
トーン: カジュアルだが専門性を感じる

ルール:
- 140文字以内（日本語）
- 冒頭に引きのあるフック
- 絵文字は1〜2個まで
- ハッシュタグは2〜3個
- CTA（リプ、RT、リンククリック等）を含める
```

**スレッド投稿の生成：**
```
以下のテーマでX（Twitter）のスレッド投稿を作成してください。

テーマ: [テーマ]
構成: 5ツイートのスレッド
1ツイート目: フック（興味を引く問いかけ）
2〜4ツイート目: 本文（具体的なノウハウ・事例）
5ツイート目: まとめ＋CTA

各ツイートは140文字以内で作成してください。
```

### Instagram向けプロンプト

**キャプション生成：**
```
Instagramのキャプションを作成してください。

投稿テーマ: [テーマ]
投稿タイプ: [フィード/リール/カルーセル]
ターゲット: [ターゲット層]

構成:
- 1行目: フック（スクロールを止める一言）
- 本文: 有益な情報を3〜5ポイントで紹介（改行を多用）
- CTA: 保存・フォロー・コメントを促す一言
- ハッシュタグ: 15〜20個（大・中・小のボリュームをミックス）

トーン: 親しみやすく、専門用語は避ける
文字数: 500〜800文字
```

**リール台本の生成：**
```
Instagram Reelsの台本を作成してください。

テーマ: [テーマ]
尺: 30秒〜60秒
構成:
- 冒頭3秒: フック（「知らないと損する〇〇」形式）
- 本編: ポイントを3つ紹介（テロップ用テキスト付き）
- ラスト: フォロー・保存を促すCTA
```

### LinkedIn向けプロンプト

**ビジネス投稿生成：**
```
LinkedInのビジネス投稿を作成してください。

テーマ: [テーマ]
ターゲット: ビジネスパーソン、経営者、マネージャー層

構成:
- 冒頭: 問題提起または実体験のエピソード
- 本文: 学びや気づきを3ポイントで展開
- 結び: 読者への問いかけ（コメントを促す）

トーン: プロフェッショナルだが堅すぎない
文字数: 800〜1,200文字
改行を多用し、読みやすくする
ハッシュタグ: 3〜5個（業界キーワード）
```

## AI投稿で気をつけるべきポイント

### やるべきこと

- **パーソナリティを加える**: AIの出力に自分の経験・意見を追加
- **事実確認**: AIが生成した数字やデータは必ず検証
- **A/Bテスト**: 異なるトーンの投稿でエンゲージメントを比較
- **トレンドの反映**: AIの提案＋リアルタイムトレンドを組み合わせる

### 避けるべきこと

- **AI文章のコピペ投稿**: 個性がなくなり、フォロワーが離れる
- **全プラットフォーム同一文**: 各SNSの文化に合わせた調整が必要
- **過度な投稿頻度**: 量より質を重視し、フォロワーのフィード汚染を避ける
- **ハッシュタグの乱用**: 特にXでは3個以内に抑える
