## 方法1：Excel Copilotでデータ分析する

### Excel Copilotとは？

Microsoft 365に統合されたAIアシスタントです。Excel上で直接AIに質問・指示ができ、数式の作成、データの分析、グラフの生成までをサポートします。

### 基本的な使い方

1. **Excelを開き、Copilotボタンをクリック**
2. **サイドパネルに日本語で指示を入力**
3. **AIが数式・グラフ・分析結果を提案**
4. **「適用」をクリックして反映**

### 実践プロンプト例

**数式の自動生成：**
```
「B列の売上金額をC列の商品カテゴリ別に合計して」
→ SUMIFSやピボットテーブルを自動生成
```

**データの傾向分析：**
```
「過去12ヶ月の売上データからトレンドを分析して」
→ 増減傾向、季節性、異常値をハイライト
```

**グラフの自動作成：**
```
「四半期ごとの売上推移を折れ線グラフで表示して」
→ 最適なグラフ形式で自動作成
```

### Copilotが得意なこと

- **VLOOKUP/XLOOKUP関数の自動生成**: テーブル結合もAIにお任せ
- **条件付き書式の設定**: 「赤字の月を赤くハイライト」で即設定
- **データクレンジング**: 重複削除、空白セル処理を自動実行
- **ピボットテーブル作成**: ドラッグ操作なしで集計表を生成

## 方法2：ChatGPTでExcel数式を生成する

Copilot Proのライセンスがなくても、ChatGPTを使えばExcelの数式を簡単に作成できます。

### 活用パターン①：数式を教えてもらう

**プロンプト例：**
```
Excelで以下の処理をする数式を教えてください。
- A列に日付、B列に売上金額が入っています
- 2026年3月のデータだけを合計したい
- 結果をD2セルに表示したい
```

ChatGPTは`=SUMPRODUCT((MONTH(A:A)=3)*(YEAR(A:A)=2026)*B:B)`のような数式を丁寧な解説付きで返してくれます。

### 活用パターン②：エラーの原因を特定する

**プロンプト例：**
```
ExcelでVLOOKUPを使ったら#N/Aエラーが出ます。
数式: =VLOOKUP(A2, Sheet2!A:C, 3, FALSE)
原因と解決方法を教えてください。
```

### 活用パターン③：データ分析の手順を聞く

**プロンプト例：**
```
以下の売上データをExcelで分析したいです。
列: 日付、店舗名、商品カテゴリ、売上金額、数量
やりたいこと: 店舗別×カテゴリ別の売上構成比を出したい
手順をステップバイステップで教えてください。
```

## 方法3：ChatGPT Advanced Data Analysisを使う

ChatGPT Plusの「Advanced Data Analysis」機能を使えば、ExcelファイルをアップロードしてAIに直接分析させることもできます。

### 手順

1. **ChatGPT Plusにログイン**
2. **Excelファイル（.xlsx）をチャットにドラッグ＆ドロップ**
3. **「このデータの売上傾向を分析して」と指示**
4. **AIがPythonコードを実行し、グラフ付きで結果を表示**

### メリット

- Excelを開く必要すらない
- 高度な統計分析（回帰分析、相関分析）も可能
- グラフをそのままダウンロードできる

### 注意点

- 機密データのアップロードは社内ポリシーを確認
- ChatGPT Team/Enterpriseプランなら学習に利用されない
