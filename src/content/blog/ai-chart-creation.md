## 方法1：ChatGPT Advanced Data Analysisでグラフを作る

### 基本手順

1. **ChatGPT Plusにログイン**
2. **データファイル（Excel/CSV）をチャットにアップロード**
3. **「このデータでグラフを作って」と日本語で指示**
4. **生成されたグラフをダウンロード**

### 実践プロンプト集

**基本的な棒グラフ：**
```
アップロードしたCSVデータで、月別売上の棒グラフを作成してください。
- X軸: 月（1月〜12月）
- Y軸: 売上金額（万円単位）
- 色: 青系のグラデーション
- タイトル: 「2025年度 月別売上推移」
```

**複合グラフ：**
```
以下の2軸グラフを作成してください：
- 棒グラフ: 月別売上金額（左軸）
- 折れ線グラフ: 前月比成長率（右軸）
- 凡例を右上に配置
- 日本語フォントを使用
```

**円グラフ：**
```
商品カテゴリ別の売上構成比を円グラフで表示してください。
- 上位5カテゴリはそれぞれ表示
- それ以外は「その他」にまとめる
- パーセンテージをラベルに表示
```

**ヒートマップ：**
```
曜日×時間帯のアクセス数をヒートマップで可視化してください。
- X軸: 時間帯（0時〜23時）
- Y軸: 曜日（月〜日）
- 色: 赤系（アクセスが多いほど濃い赤）
```

### ChatGPTグラフ作成のコツ

- **色の指定は具体的に**: 「コーポレートカラーの#003366を基調に」のように指定
- **フォントサイズの調整**: 「タイトルは16pt、軸ラベルは12ptで」
- **複数グラフの一括作成**: 「以下の4種類のグラフをサブプロットで1枚にまとめて」
- **反復改善**: 「もう少し余白を広くして」「凡例の位置を変えて」と修正指示

## 方法2：Claudeアーティファクトでインタラクティブグラフを作る

### Claudeアーティファクトの特徴

ClaudeのアーティファクトはHTML/JavaScriptでグラフを生成するため、**マウスホバーで数値表示、ズーム、フィルタリングなどのインタラクティブ操作**が可能です。

### 基本手順

1. **Claude.aiにアクセス**
2. **データを貼り付けるか、ファイルをアップロード**
3. **「アーティファクトでグラフを作って」と指示**
4. **ブラウザ上で操作可能なグラフが生成される**

### 実践プロンプト集

**ダッシュボード作成：**
```
以下の売上データでダッシュボードを作成してください：
- 上部: KPIカード（総売上、前月比、目標達成率）
- 中央: 月別売上推移の折れ線グラフ
- 下部左: カテゴリ別構成比の円グラフ
- 下部右: 支店別売上ランキングの棒グラフ

データ:
[データをここに貼り付け]
```

**フィルター付きグラフ：**
```
売上データで地域別にフィルタリングできるグラフを作ってください。
ドロップダウンで地域を切り替えると、
その地域の月別売上推移が表示されるようにしてください。
```

**アニメーション付きグラフ：**
```
年度別の売上推移をアニメーションで表示するグラフを作ってください。
「再生」ボタンを押すと、2020年から2025年までの
データが順番に表示される形式で。
```

### Claudeアーティファクトの活用シーン

- **経営会議用ダッシュボード**: 複数のKPIを1画面に集約
- **プレゼン資料の補助**: インタラクティブデモとして使用
- **データ探索**: フィルタやドリルダウンで深掘り分析

## 用途別おすすめ使い分けガイド

### レポート・資料用の静的グラフ → ChatGPT

PowerPointやPDF資料に貼り付ける**画像グラフ**が必要な場合はChatGPTが最適です。高解像度のPNG/SVGでダウンロードでき、そのまま資料に使えます。

### プレゼン・ブラウザ表示用 → Claude

ライブデモやブラウザで見せる**インタラクティブグラフ**にはClaudeが最適です。ホバー表示やフィルタリングで、データの深掘りが可能です。

### 定期レポートの自動化 → Google Sheets / Excel

毎月同じ形式のレポートを作る場合は、**テンプレート化してGemini/Copilotで更新**するのが効率的です。
